# AI Tutor Bot

**AI Tutor Bot** — Telegram-бот для анализа учебных PDF-документов с использованием искусственного интеллекта.

После получения PDF-файла бот извлекает из него текст, определяет основные идеи и с помощью модели **Google Gemini 2.5 Flash** формирует структурированный конспект. Кроме этого, пользователь получает краткую статистику документа: количество символов, слов и примерное время чтения оригинального файла.

Проект разработан в рамках кейсового задания для поступления на образовательную программу **«Технологии искусственного и дополненного интеллекта»**.

---

# Какую проблему решает проект?

При подготовке к экзаменам и занятиям студентам приходится работать с большим количеством учебных материалов. Чтение объемных лекций и пособий занимает много времени, а найти основные мысли бывает непросто.

AI Tutor Bot помогает быстро ознакомиться с содержанием документа, выделяя наиболее важную информацию.

Проект может быть полезен:

- студентам
- школьникам старших классов
- преподавателям
- всем, кто работает с учебными PDF-документами.

# Запуск проекта

## 1. Клонировать репозиторий

```bash
git clone https://github.com/danya-sever/AI-tutor-bot.git
```

## 2. Перейти в папку проекта

```bash
cd AI-tutor-bot
```

## 3. Создать виртуальное окружение

```bash
python -m venv .venv #(python3 для Mac)
```

## 4. Активировать виртуальное окружение

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

## 5. Установить зависимости

```bash
pip install -r requirements.txt
```

## 6. Создать файл `.env`

```env
BOT_TOKEN=Ваш_токен_Telegram_бота
GEMINI_API_KEY=Ваш_API_ключ_Google_Gemini
```

## 7. Запустить проект

```bash
python bot.py
```

# Роль искусственного интеллекта

В проекте используется языковая модель **Google Gemini 2.5 Flash**.

После извлечения текста из PDF бот формирует промпт и отправляет его через API в модель Gemini.

Нейросеть анализирует содержание документа и формирует структурированный конспект, включающий:

- тему
- основные идеи
- ключевые термины
- краткий вывод.

Использование ИИ позволяет значительно сократить время знакомства с большим объемом учебного материала.

---

# Ограничения проекта

- поддерживаются только PDF-файлы
- требуется подключение к интернету
- необходим действующий API-ключ Google Gemini
- качество конспекта зависит от качества исходного документа
- очень большие документы могут быть обработаны не полностью из-за ограничений модели.
